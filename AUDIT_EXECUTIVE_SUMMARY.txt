================================================================================
SCHEMA IMPLEMENTATION DETAILS
================================================================================

REQUIRED SCHEMA TYPE: BlogPosting
PLACEMENT: Immediately BEFORE the <div class="blog-post"> tag
NOT INSIDE the blog-post div

TEMPLATE STRUCTURE:
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "BlogPosting",
  "headline": "[POST TITLE]",
  "description": "[BRIEF DESCRIPTION/EXCERPT]",
  "image": "[IMAGE URL IF AVAILABLE]",
  "author": {
    "@type": "Person",
    "name": "BetLegend Picks"
  },
  "publisher": {
    "@type": "Organization",
    "name": "BetLegend Picks",
    "logo": {
      "@type": "ImageObject",
      "url": "https://www.betlegendpicks.com/newlogo.png"
    }
  },
  "datePublished": "[PUBLICATION DATE IN ISO 8601]",
  "dateModified": "[LAST MODIFIED DATE IN ISO 8601]"
}
</script>

VALIDATION: After implementation, test each schema using:
- Google Rich Results Test: https://search.google.com/test/rich-results
- Schema.org Validator: https://validator.schema.org/
